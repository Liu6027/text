# STALAB

![STALAB Banner](image.png)

STALAB 专注于智能硬件与计算机视觉方向的产品研发，围绕低功耗显示、边缘感知与场景智能化持续构建可落地的技术方案。

我们关注从设备、算法到系统集成的完整链路，致力于将技术能力转化为真实场景中的可部署产品，为信息展示与安全监测提供更稳定、高效的解决方案。

## 我们在做什么

### 墨水屏广告牌

面向低功耗、长续航和高可视性的展示场景，构建基于墨水屏的智能广告牌系统。项目聚焦内容分发、远程管理、显示控制与终端部署，适用于校园、园区、门店及公共信息展示等场景。

### 视觉火灾报警

基于计算机视觉技术构建火灾早期识别与告警系统，重点关注烟雾、火焰等异常目标检测，以及复杂环境下的识别稳定性与实时响应能力，适用于实验室、仓储及园区安全监测场景。
