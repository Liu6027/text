## About STALAB
STALAB 专注于智能硬件与计算机视觉方向的产品研发，围绕低功耗显示、边缘感知与场景智能化持续构建可落地的技术方案。
我们关注从设备、算法到系统集成的完整链路，致力于将技术能力转化为真实场景中的可部署产品，为信息展示与安全监测提供更稳定、高效的解决方案。
我们相信，技术的价值不仅在于“做出来”，更在于“跑得稳、落得下、用得起来”。

## Projects
### E-Ink Billboard
[Project Repository](https://github.com/ScineceTechAILab/2026-E_Ink_Billboard_Backend.git)
面向低功耗、长续航、高可视性的展示需求，我们正在构建基于墨水屏的智能广告牌系统。项目聚焦内容分发、远程管理、显示控制与终端部署，适用于校园、园区、门店及公共信息展示等场景。
这不仅是一块“会显示内容的屏幕”，更是一套可远程运营、可规模部署、可持续维护的智能展示基础设施。
### Visual Fire Detection
[Project Repository](https://github.com/ScineceTechAILab/2025-Fire_Detection_System.git)
基于计算机视觉技术，我们构建面向火灾早期识别与告警的智能系统，重点关注烟雾、火焰等异常目标检测，以及复杂环境下的识别稳定性与实时响应能力。
项目适用于实验室、仓储、园区等安全监测场景，致力于让风险更早被发现，让响应更快被触发，让安全防线更智能。
